About HistoryNear.me

Welcome to HistoryNear.me, a site to help you find a little bit of history near by. We are still in beta testing right now and are only utilizing data around San Diego, California. If you have any questions or comments, leave them here.

How

  1. Use the geo-locate button and we'll try to use your current location, or use the address search form to create a starting point.
  2. Internet magic will happen and we'll try to find the closest bit of history to you.
  3. Utilize the walking directions to start exploring.

Technology

Map imagery from Mapbox; Map data © OpenStreetMap contributors, CC-BY-SA; Directions courtesy of MapQuest ; Maps powered by Leaflet; Interface styling based from Bootstrap; Icons provided by map icons; Database hosting by CartoDB. Get the code on Github.

Data

This application is powered by open data provided by the City of San Diego.
